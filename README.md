## Installation 💻

Make sure that you have [Node.js 16.0 version or higher](https://nodejs.org) installed on your machine 📠.

When you are in the root 🌱 folder 📁, install all dependencies using npm:

```bash
$ npm install
```

Notice that this project using [Vite](https://vitejs.dev) ⚡ instead of [Vue CLI](https://cli.vuejs.org/) because it is in Maintenance Mode 😴 and [official Vue CLI website](https://cli.vuejs.org/) recommends new projects to use Vite.

## Usage 🕹️

You can run the dev server 🖥:

```bash
$ npm run dev
```

Or you can build ⚒️ the project and look at the preview 🖼️:

```bash
$ npm run build
# And then, when it finishes building
$ npm run preview
```

If you open package.json 📦 file, you might see there is a [gulp](https://gulpjs.com) 🥤 with it's plugins in "devDependencies" field.

In that moment, gulp 🥤 is only used to convert .ttf format fonts contained in path "@/assets/fonts" to .woff format for legacy browsers 🕚 and .woff2 format for modern browsers 🌐.

Fonts automatically adds to fonts.scss 🅰 file and imports to style.scss 🎨 file.
